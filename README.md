# Phase10 Android App (Vorschau)

Eine einfache Compose-Vorschau-App für Phase 10 mit:
- Spieler- und Admin-Anmeldung (inkl. Gastmodus)
- Verwaltung von Punkten (1 Punkt = 1 Cent Abzug vom eingezahlten Betrag)
- Phasenverwaltung mit frei editierbarer Info-Spalte
- Darstellung von Guthaben, aktueller Phase und Rangliste

> Hinweis: Diese Codebasis ist ein UI-Prototyp ohne Persistenz oder Netzwerk. Für eine produktive App müssen z. B. Datenbank, Authentifizierung und Synchronisation ergänzt werden.

## Start
Dieses Projekt nutzt Android Gradle Plugin 8.2.2 und Jetpack Compose. Öffne den Ordner in Android Studio (Koala oder neuer), lasse die Gradle-Synchronisation laufen und starte die App auf einem Emulator oder Gerät (minSdk 24).
